Vector Fields & Flow

How do we describe where a fluid is going at every point in space? We use vector fields: a collection of arrows showing direction and speed.

What is a Vector?

A vector is simply a quantity that has both a magnitude (size/length) and a direction.

Vector (Simple Definition)

Think of a vector as an arrow: it points somewhere (direction) and has a length (magnitude). We write vectors as $\mathbf{v} = (v_x, v_y)$ where $v_x$ is the horizontal component and $v_y$ is the vertical component.

For fluid flow, we care about velocity vectors: at each point in space, the vector tells us which way the fluid is moving and how fast.

What is a Vector Field?

A vector field assigns a vector to every point in space. Imagine placing a tiny arrow at every location‚Äîthat's a vector field!

  • Uniform: Same flow everywhere (like wind on a calm day)

  • Source: Fluid spreading out from a point (like a fountain)
  • Sink: Fluid collecting toward a point (like a drain)

Streamlines, Pathlines, and Streaklines

There are different ways to visualize flow:

Streamlines

Curves that are everywhere tangent to the velocity field at a single instant. Think of them as the paths fluid would take if the flow were frozen in time.

Pathlines

The actual path a single particle traces as it moves through the fluid over time. The particles in our demo trace pathlines!

Streaklines

If you injected dye at a point, the streakline is where all the dyed particles would be at a given moment.

üí° Steady vs. Unsteady Flow

For steady flow (where the field doesn't change over time), streamlines, pathlines, and streaklines are all the same! Our demo shows steady flow for clarity.

Why Vector Fields Matter

Vector fields are the language we'll use to describe everything in this course:

  • Divergence will measure how much the field "spreads out"
  • Curl will measure how much the field "rotates"
  • Line integrals will add up the field along paths
  • Stokes' Theorem will connect all these ideas beautifully!
